# Code of Conduct

## Standard

This project operates under the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1.

## Summary

Be direct, be respectful, focus on technical merit. Personal attacks, harassment, and bad-faith arguments are not acceptable in any form.

## Scope

Applies to all project spaces: issues, PRs, discussions, code review comments, and any communication representing this project.

## Enforcement

Violations can be reported to the project maintainers. Maintainers will respond promptly and handle reports confidentially.
